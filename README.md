# Healthcare Real-World Data Analysis & Prediction Project

## Project Overview

This project focuses on analyzing a real-world healthcare dataset using data science and machine learning techniques. The project includes data cleaning, exploratory data analysis (EDA), visualization, and predictive modeling to uncover healthcare trends and insights.

The goal is to apply end-to-end data science workflows in a real-world healthcare domain.

---

## Objectives

* Clean and preprocess healthcare data
* Perform exploratory data analysis
* Identify trends and correlations
* Visualize healthcare insights
* Build machine learning prediction models
* Evaluate model performance
* Generate structured analytical reports

---

## Dataset Used

Dataset:

* `healthcare_dataset.csv`

The dataset contains healthcare-related patient information such as:

* Age
* Gender
* Medical details
* Admission and discharge dates
* Billing amount
* Hospital-related information
